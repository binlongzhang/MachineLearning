# Machine Learning

一些基础机器学习算法实现

主要参考

> 李锐, 李鹏, 曲亚东, 等. 机器学习实战[M]. BEIJING BOOK CO. INC., 2013.

使用纯python3实现，没有使用相关的框架，修改，补充，完善了参考资料中的相关细节，所有实验数据都在data文件夹下；

代码和原理均以Jupyter Lab的形式展示，方便阅读和学习；
